Como Doar

O processo de doação de sangue é simples e rápido. Veja abaixo como você pode contribuir para salvar vidas:

1. Requisitos para Doar

Para doar sangue, é preciso atender a alguns requisitos. Confira se você se enquadra:

  • Estar saudável
  • Pesar mais de 50kg
  • Ter idade entre 16 e 69 anos

2. O que acontece após a doação?

Após doar, você será orientado sobre os cuidados necessários e poderá retomar suas atividades normalmente após um breve período de repouso.
